# CITADEL Website

#### `jekyll` - Quickstart if you are new

```sh
# Start by installing ruby and prerequisites (https://jekyllrb.com/docs/installation/)
gem install jekyll bundler # Install the jekyll and bundler gems
jekyll new myblog && cd myblog # Create a new Jekyll site
bundle exec jekyll serve --livereload # bundle exec jekyll serve
```
